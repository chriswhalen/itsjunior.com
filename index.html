junior

,

a full stack web framework

junior is driven by Flask and provides an opinionated set of tools to efficiently build a performant web application.

It requires Python 3.7+ with pip; PostCSS support requires Node.js 12.0+ with npm.

Read the docs View on GitHub

getting started

Install junior: 

$ pip install junior

Create app.py: 

from junior import Application
application = Application()
application.start()

Start flask: 

$ flask run

All done! You can find your new app at http://localhost:8000/.
